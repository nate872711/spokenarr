# Spokenarr

**Spokenarr** is an automated audiobook manager with download integration, metadata organization, and library tracking.

Frontend: **React + Storybook** (main UI)
Backend: **FastAPI**
Database: **PostgreSQL**
Dockerized for easy setup.
